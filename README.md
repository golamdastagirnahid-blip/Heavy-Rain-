# 🌧️ Heavy Rain Deep Sleep - YouTube Automation

Automatically creates and uploads heavy rain sleep videos to YouTube.

## How It Works

1. Downloads audio from Google Drive
2. Gets rain footage from Pexels/Pixabay
3. Loops footage over audio
4. Generates AI titles/descriptions
5. Creates thumbnails
6. Uploads to YouTube
7. Runs 3 times per day automatically

See SETUP.md for full instructions.

## Secrets Required

- `GH_TOKEN` - GitHub token
- `YOUTUBE_CLIENT_SECRETS` - YouTube API credentials
- `YOUTUBE_TOKEN` - YouTube OAuth token
- `PEXELS_API_KEY` - Pexels API key
- `PIXABAY_API_KEY` - Pixabay API key
- `OPENROUTER_API_KEY` - OpenRouter API key
